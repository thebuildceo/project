# Screenshots Directory

This directory should contain screenshots of the application for the README.md file.

## Required Screenshots

1. **home_page.png** - Screenshot of the home page showing the dashboard overview
2. **data_analysis.png** - Screenshot of the data analysis page with visualizations
3. **prediction.png** - Screenshot of the prediction page with a sample prediction
4. **model_insights.png** - Screenshot of the model insights page showing feature importance

## How to Capture Screenshots

1. Run the application locally:
   ```bash
   streamlit run app.py
   ```

2. Navigate to each page in the application

3. Take a screenshot of each page

4. Save the screenshots in this directory with the names listed above

## Note

The screenshots are referenced in the main README.md file. Make sure the filenames match exactly.
